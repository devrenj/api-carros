# api-carros
API de veículos automotivos para consulta

### Requerimentos

Uma das versões do [Node.js](https://nodejs.org/en/download/) deve estar instalada para executar o comando **`npm`**:

* `18.x.x` >= `18.17.0`
* `20.5.0` ou atual

### Instalando as dependências

```bash
npm install
```
